# AdventureWorksDW2025-FPA-Analysis

## Project Overview

This project demonstrates a complete Financial Planning & Analysis (FP&A) workflow using the AdventureWorksDW2025 sample data.

The project covers:
- Data sourcing from SQL Server using a .bak restoration.
- Data modelling using a star schema.
- Financial data transformation.
- Excel FP&A modelling techniques for forecasting and variance analysis.
- Version control using GitHub.

## Project Architecture

Data Source:  SQL Server - AdventureWorksDW2025

Data Transformation: SQL Views - Fact/Dimension Modelling

Analysis Layer: Excel FP&A Model

Version Controls: GitHub Repository

## Objectives

- Build a data warehouse-based FP&A model
- Perform revenue and cost analysis
- Conduct variance and trend analysis
- Create a scalable Excel FP&A dashboard
